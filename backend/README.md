## 📸 项目简介

**公元胶片模拟器** 是一个基于 **FastAPI + OpenCV** 的智能图像滤镜 Web 应用。它通过精确模拟 8 款经典公元胶片的**色彩、反差、颗粒和纸基质感**，让你的数码照片穿越时光，重现胶片时代的独特温度。

项目灵感来源于**汕头公元感光材料厂**（1953-2003）——中国第一家感光材料厂，也是中国感光工业的摇篮。所有滤镜参数均基于历史史料、实物样片和当年的技术档案校准。

## 📜 每一款胶片背后，都有一段中国故事

### 🎞️ 公元黑白胶卷 1980s
> *"那是一整个朴素而热气腾腾的年代。"*

1958年，汕头公元厂的第一卷黑白胶卷走下生产线。在此之前，中国人用的胶卷全部依赖进口——柯达叫"矮克发"，富士叫"樱花牌"，一卷胶卷抵得上普通工人一周口粮。1981年，公元黑白胶卷拿下国家银质奖章，从此成为中国家庭记录人生大事的标配：满月照、毕业照、全家福、探亲团聚……你在任何一本80年代的家庭相册里，都能找到它的影子。

### 👤 公元人相胶片
> *"人像片没有厂子可以跟我们竞争。"*

1955年4月，工程师林希之在汕头的小作坊里，试制成了中国第一张人像胶片。这是一种4×5英寸的大画幅散页片，专门供给照相馆的木制座机使用。从建国初的干部标准像，到文革后的第一次结婚登记照，再到80年代的艺术照潮——你祖父母、父母身份证上的那张照片，很大概率就是用公元人像胶片拍的。

### 🎨 公元彩色负片 1985
> *"一盒彩卷带冲印要三十多块，相当于普通工人三天的工资。"*

1984年，汕头公元厂第一次从日本富士买进了整箱的彩色胶卷"大轴"——一人高、几公里长的母卷。工人们在恒温恒湿的暗房里分切、装卷、贴标签。1985年产量达到三百万卷，供不应求。中国老百姓终于不用再只拍黑白照了，春游、生日宴、第一次去北京天安门，都变成了带颜色的记忆。

### 🔮 公元彩色反转片
> *"拉上窗帘，架起幻灯机，咔哒一声切换下一张。"*

彩色反转片又叫"幻灯片"——拍出来的是正像，对着灯光看，每一张都通透得像彩色玻璃。1980年代末，公元引进油溶性工艺，解决了早期反转片"见光掉色"的问题。在那个没有 PowerPoint 的年代，大学教室、单位会议室、旅行社的线路说明会……这是中国人最着迷的"沉浸式放映体验"。

### 🎬 公元彩色电影正片 1970s
> *"那是整个村子一年中最热闹的夜晚。"*

1973年，北京电影制片厂用公元牌彩色电影正片印出来的拷贝，专家们看完后一致评价："与美国伊士曼柯达胶片相比，没有明显差别。"同年，这款胶片拿下全国科学大会优秀科技成果奖。1970年代的夏夜，村口晒谷场架起白布，放映员手摇发电机启动——公元电影胶片在放映机里咔哒咔哒地转动，那是整个村子一年中最热闹的夜晚。

### 🦴 公元X光胶片
> *"它不记录美，它记录结构、裂纹和真相。"*

很多人不知道，公元厂除了民用胶卷，还是中国医用和工业X光胶片的核心供应商。蓝色片基，双面涂布乳剂——才能在X光下捕捉到钢板的裂纹、人体的骨骼。1960-90年代，国内医院的每一张胸片、每一张骨科透视片、航空航天工厂每一块机翼铸件的无损检测——几乎都印着公元的牌子。

### 🖨️ 公元印刷制版胶片
> *"没有它，印不出一张报纸。"*

1964年，公元牌PB中性全色制版胶片拿下国家奖项。在此之前，印刷制版胶片全靠进口，一张大报的付印可能因为缺几张胶片而延期。制版胶片是"黑白分明"到了极致的东西——只有纯黑和纯白，没有任何灰阶。报纸上的照片、书上的插图、烟盒上的图案——所有印刷品，都必须先用它拍出"网点底片"才能上印刷机。

### 📜 公元黑白印相纸
> *"暗房里永远安静，只有流水声和镊子碰盘子的叮当。"*

1952年，汕头公元的实验室里诞生了中国第一张符合使用要求的黑白相纸。在此之前，暗房用的全是德国"矮克发"和英国"伊尔福"。公元印相纸分四个反差等级（1号软→4号特硬），纸基分光面、绸纹、绒面、半光四种。1985年，年产量达到120.9万盒，能绕地球大半圈。

## 🚀 快速启动

### Windows 用户（推荐）
双击项目根目录的 **`🎞️_启动公元胶片模拟器.bat`**

脚本会自动：
1. 检测 Python 环境
2. 安装所有依赖（使用清华镜像加速）
3. 启动后端服务
4. 自动打开浏览器

### 手动启动
```bash
# 1. 进入后端目录
cd backend

# 2. 安装依赖
pip install -r requirements.txt

# 3. 启动服务
python main.py
